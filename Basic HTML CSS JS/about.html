🎯 CSS Position Cheat Sheet

🔵 Relative: Shifts visually, but keeps its original space in the flow.

🔴 Absolute: Leaves normal flow. Positions itself relative to the nearest positioned parent (the dashed box).

🟢 Fixed: Glued to the viewport. Never moves when scrolling.

🟡 Sticky: Acts like relative until you scroll past its top value, then locks in place.
